# Tshidi's Kitchen
My POE part 1

-Project Overview
This is the official website for TSHIDI'S KITCHEN, a South African food business specializing in authentic Limpopo cuisine.
 The website showcases their products, tells their story, and provides ways for customers to connect with the business.

## Features
    - Contact form with validation
    - Product menu with pricing
    - Embedded Google Map for location
    - Embedded Google Form for enquiries
